# Produktvision

StageBase ist ein internes Ressourcenplanungssystem fuer Veranstaltungstechnik. Es buendelt Jobs, Crew, Dienstplan, Material und spaeter Fahrzeuge in einer gemeinsamen Planungsoberflaeche.

Die Anwendung ist fuer die eigene Organisation gedacht. Es gibt keine oeffentliche Registrierung und keinen externen SaaS-Mandantenbetrieb.

## Zielgruppen

- Disposition
- Projektleitung
- Lager und Materialplanung
- Crew-Koordination
- spaeter mobile Crew-Nutzer

## Langfristig geplante Module

- Dashboard
- Jobs
- Dienstplan
- Anfragen
- Crew
- Inventar
- Packlisten
- Fahrzeuge
- Kalender
- Benachrichtigungen
- Einstellungen

## Produktprinzip

Alles, was zeitlich geplant werden kann, wird als Resource gedacht. Personal, Inventar, Fahrzeuge, Cases, Verbrauchsmaterial und Raeume nutzen denselben Ressourcen-Kern.
