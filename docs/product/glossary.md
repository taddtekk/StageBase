# Glossar

## Job

Eine Veranstaltung oder ein Auftrag mit Zeitraum, Ort, Anforderungen und geplanten Ressourcen.

## Schicht

Ein geplanter Arbeitszeitraum fuer Crew oder interne Rollen.

## Crew

Interne oder eingeplante Personen, die als Ressourcen geplant werden koennen.

## Resource

Alles, was zeitlich geplant werden kann: Mitarbeiter, Asset, Fahrzeug, Case, Verbrauchsmaterial oder Raum.

## Requirement

Eine Anforderung an eine Resource oder einen Resource-Typ fuer einen Job oder Planungszeitraum.

## Booking

Eine konkrete Reservierung oder Buchung einer Resource fuer einen Zeitraum.

## Asset

Ein technischer Gegenstand oder eine Materialposition, die geplant werden kann.

## Case

Ein Transport- oder Packelement, das als eigene Resource vorbereitet ist.

## Packliste

Eine Liste geplanter oder gebuchter Ressourcen fuer Vorbereitung, Ausgabe und Ruecklauf.
